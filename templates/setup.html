Email Notifications Setup

To receive booking notification emails, you need to configure email settings:

  1. Set MAIL_USERNAME (your email address)
  2. Set MAIL_PASSWORD (your app password)
  3. Optionally configure MAIL_SERVER and MAIL_PORT

Note: The system works without email configuration - bookings will still be saved and viewable in the admin panel.
